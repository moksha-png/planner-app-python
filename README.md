# Planner App (Python)

A simple **planner app** built in Python to **add, view, mark complete, and delete tasks**.  
This project helps organize daily tasks in a simple and interactive way.

# Features
- Add new tasks
- View all tasks
- Mark tasks as complete
- Delete tasks
- Save/load tasks (optional, depending on implementation)

# How to Run

1. Open the app in Python IDE or compiler.
2. Run the `planner.py` file.
3. Follow the on-screen menu to manage tasks:
   - Add task
   - View tasks
   - Mark task complete
   - Delete task
